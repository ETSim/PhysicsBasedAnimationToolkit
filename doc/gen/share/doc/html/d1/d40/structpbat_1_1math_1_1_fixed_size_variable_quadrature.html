Detailed Description

template<class Quad>
struct pbat::math::FixedSizeVariableQuadrature< Quad >

Represents a quadrature scheme that can be constructed via existing quadrature schemes.

However, this generic quadrature scheme can be modified, i.e. its points and weights are instance member variables.
